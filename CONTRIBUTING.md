# Contributing

Thanks for helping build Cosma. This project is a pre-alpha prototype, so the
focus is on small, well-tested iterations.

Getting started
- Go 1.22+ installed.
- `go test ./...`
- `golangci-lint run` (install via https://golangci-lint.run)

Workflow
- Keep PRs small and focused.
- Include tests for new behavior.
- Run `go test ./...` before opening a PR.
- Use `gofmt` on changed Go files.

Questions
If you are unsure about a change, open an issue or start a draft PR for early
feedback.
