# Code of Conduct

We want Cosma to be a welcoming and inclusive project.

Our standards
- Be respectful and considerate.
- Assume good intent and provide constructive feedback.
- No harassment, discrimination, or abusive behavior.

Scope
This code of conduct applies to project spaces and public communication around
Cosma.

Reporting
If something happens that violates this code of conduct, please reach out to the
project maintainer via the contact listed in `SECURITY.md`.
